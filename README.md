# skill-dev

## 用途
这个仓库用于维护一组独立的 skill 文档。每个顶层目录代表一个单独的 skill，仓库本身保持轻量，不承载构建系统、应用代码或发布产物。

## 目录
- `brainstorming-ralph/`：Ralph 工作流相关的头脑风暴 skill。
- `doc-code-aligner/`：文档与代码对齐分析 skill，包含 `references/` 辅助材料。
- `repo-architect/`：用于分析仓库结构与架构的 skill。
- `roadmap-dev/`：轻量路线图规划 skill。
- `AGENTS.md`：仓库贡献说明。

## 维护原则
只在必要时增加文件和目录，保持每个 skill 自包含。仓库以 Markdown 文档为主，结构应清晰、直接，避免引入不必要的脚本和工具配置。
